HTML

HTML stands for Hypertext Markup Language . It is the standdard markup language for creating web pages and web applications. WEb browsers receive HTML documents from a web server of from local storage and render them into multimedia web pages.

HTML describes the structure of a web page semantically and originally included cues for the appearance of the document. HTML elements are the building blocks of HTML pages.

HTML Tags

BR TAG
BR TAG is used to create a line break between the first and second lines of the paragraph and it doesn't need a closing tag.

LI TAG
The LI TAG is an HTML tag used to define a list item within an ordered list (OL TAG) or an unordered list (UL TAG). LI TAG is a child tag of OL TAG and UL TAG.

IMG TAG
The IMG TAG stands for the image tag. We can put some images on our web by using this tag. IMG TAG has attributes. It is important to note that the img tag must include the src attribute (it stands for source), as it specifies the location of the image file. Without the src attribute, the image will not be displayed on the web page.
ex) src="file name or file location"

The alt attribute(it stands for alternative) is important for accessibility, as it provides a text description of the image that can be read by screen readers or displayed in place of the image if it cannot be loaded.

The img tag also supports several other attributes, including:

  • width : Specifies the width of the image in pixels or as a percentage.
  • height: Specifies the height of the image in pixels or as a percentage.
  • title: Provides additional information about the image, which can be displayed as a tooltip when the user hovers over the image.
  • border: Specifies the width of a border around the image.

A TAG
The A TAG is an HTML tag used to define a hyperlink, or a link to another web page or resource. The A tag is an inline element, which means it can be used within a line of text or other content. A stands for Anchor and its meaning is Hypertext.

In this example, the href attribute specifies the URL of the web page or resource to link to, and the text "Click here to visit Example.com" is the visible text that the user will see and click on to follow the link.Its meaning is Hypertext Reference.

  • target: Specifies where to open the linked document, such as in a new window or tab.
  • rel: Specifies the relationship between the current document and the linked document, such as whether it is a pre-fetch or a pre-render link.
  • title: Provides additional information about the linked document, which can be displayed as a tooltip when the user hovers over the link.
